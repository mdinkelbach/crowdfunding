When visit site
Go to homepage, has blog posts (if they exist), has nav links for homepage and dashboard, and login*

When click homepage
Goes to homepage*

When click dashboard when not logged in
Goes to login page (with option to sign up)*

When choose to sign up
Prompted to create username and password*

When click submit on sign up
Username and password saved, then user is logged in*

When revisit site later, and click log in
Asks for username and password*

When signed in
Still see nav links for homepage and dashboard, but login becomes logout*

When click homepage
Taken to homepage, shown existing posts with title and creation date*

When click on existing post
Presented with title, contents, creator's name, and creation date for that post, also can leave comment (partial *)

When enter a comment and click submit while signed in
Comment is saved and post is updated to display the comment, the comment creator's name, and date created

When click on dashboard in nav
Taken to dashboard and shown any posts user has previously made, and option to add new post

When click add new post button
Prompted to enter title and contents for post

When click create new post button
Previously entered title and contens of that post are saved, then user taken back to dashoard that has new post added to it

When click existing post in dashboard
Able to delete or update post, then taken back to updated dashboard based on which option choosen

When click logout
signed out of site*

When idle for more than a set time (30 minutes)
Auto logged out
